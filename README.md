# autocomplete-endereco
Neste projeto foi criado um autocomplete de endereço com HTML CSS e JavaScript , consultado uma API passando um CEP de algum endereço, e recebendo um JSON como retorno. Extraindo os dados do JSON podendo assim preencher os campos de: Rua, Cidade, Bairro e Estado do usuário. Este projeto é visto principalmente em e-commerces, na etapa de configuração de endereço de entrega.
